# calc--test-prj
Сервис "calc" тестового проекта calculator-v2-experimental (см. репозиторий calculator-documentation)

## Конфигурация
1. В корне добавить файл config.py
2. В config.py добавить строку "CONTROLLER_SOCKET = {socket_number}", где вместо {socket_number} нужно вставить номер нужного сокета.
